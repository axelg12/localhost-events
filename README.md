# Localhost Events

This project is a simple view on top of the JSON / RSS feed of upcoming
programming and tech related events. See that repository
[here](https://apis-is.koddsson.deno.net/x/meetups?pretty=true)

To add new events you can fill out
[this form](https://github.com/koddsson/apis.is/issues/new?template=add-meetup.yml)

Contribution are welcomed.

### Usage

Make sure to install Deno:
https://docs.deno.com/runtime/getting_started/installation

Then start the project in development mode:

```
deno task dev
```

This will watch the project directory and restart as necessary.
